# API Practice

API made for practice reasons. 

## Endpoints

- /api/persons/: The full persons API
- /api/persons/id: The element selectioned with the id

## Methods

- GET (/api/persons/ & /api/persons/id)
- POST (/api/persons)
- DELETE (/api/persons/id)

## Tech Stack

**Client:** HTML

**Backend:** NodeJS, ExpressJS
